# 🐢 Turtle Race Game

A fun, interactive betting game built with Python's **Turtle Graphics** library. This project demonstrates basic game logic, user input handling, and OOP principles.

## How to Play
1. Run the script.
2. A pop-up will ask you to place a bet: pick a color (red, orange, yellow, green, purple, or blue).
3. Watch the turtles race!
4. The terminal will announce if your turtle won or lost.

## Program Requirements
* **Python 3.6**
* The `turtle` module (included in the Python Standard Library).

## Project Structure
* `main.py`: The primary game script containing the turtle setup and racing loop.
